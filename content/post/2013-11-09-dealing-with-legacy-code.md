> 處理 Legacy Code 的步驟就是：（1）決定更新的邊界（2）依據邊界範圍設計介面（3）實作新介面的轉接器實作（4）把客戶端程式碼升級到新的介面版本（5）依計畫將轉接器實作更新為全新實作。 
>
>這提供了演化的可能，也是我最喜歡的軟體開發方式，它降低風險，而且在短期內可驗證效果，也鼓舞每個人有信心繼續做下去。
